# Code of Conduct

This project adopts the Contributor Covenant Code of Conduct.

Expected behavior:
- Be respectful and constructive.
- Focus on technical issues and facts.
- Assume good intent and communicate clearly.

Unacceptable behavior:
- Harassment, discrimination, or personal attacks.
- Publishing private information without consent.
- Repeated disruptive behavior after moderation requests.

Reporting:
- Contact `conduct@codewithphone.com`

Project maintainers are responsible for clarifying and enforcing this code of conduct.
